## 一、四元数的核心理解

### 1.1 四元数公式与模长
四元数表示旋转：  
\[
q = \left(\cos\frac{\alpha}{2},\; u_x \sin\frac{\alpha}{2},\; u_y \sin\frac{\alpha}{2},\; u_z \sin\frac{\alpha}{2}\right)
\]
- 自动满足 \( \|q\|=1 \)，因为 \(\cos^2 + \sin^2 \cdot (u_x^2+u_y^2+u_z^2)=1\)。
- 第一个分量 \(\cos(\alpha/2)\) 编码旋转角度（半角），后三个分量是 **缩放后的旋转轴**（轴分量乘以 \(\sin(\alpha/2)\)）。
- 提取旋转轴：归一化后三个分量即得单位轴。

### 1.2 四元数既可描述姿态，也可描述旋转
- **姿态**：从参考系（惯性系）到本体坐标系的旋转。
- **旋转**：对向量或另一姿态施加的转动。
- 数学上完全等价：一个单位四元数 = 一个旋转 = 一个相对定向。

### 1.3 旋转向量公式
将向量 \( \mathbf{v} \) 用四元数 \( q \) 旋转：
\[
\mathbf{v}' = q \otimes (0,\mathbf{v}) \otimes q^*
\]
其中 \( q^* \) 是共轭。运算顺序：先左乘 \( q \)，再右乘 \( q^* \)。

### 1.4 旋转复合（姿态更新）
- 当前姿态 \( q_{\text{cur}} \)，施加一个新旋转 \( q_{\text{rot}} \)（在惯性系中定义）：
  \[
  q_{\text{new}} = q_{\text{rot}} \otimes q_{\text{cur}}
  \]
- 在运动学积分中，微小旋转由角速度构造，**左乘**更新。

## 二、刚体动力学与运动学（三维）

### 2.1 角速度定义
- \( \boldsymbol{\omega} = (\omega_x, \omega_y, \omega_z) \)：在本体坐标系中测量，分别表示绕本体 X、Y、Z 轴的瞬时角速度。

### 2.2 动力学方程（欧拉方程）
\[
\mathbf{I} \dot{\boldsymbol{\omega}} + \boldsymbol{\omega} \times (\mathbf{I}\boldsymbol{\omega}) = \boldsymbol{\tau}
\]
- 若惯量矩阵与主轴对齐：\( \mathbf{I} = \mathrm{diag}(I_x, I_y, I_z) \)。
- 展开得耦合项（陀螺力矩）：
  \[
  \begin{aligned}
  I_x\dot{\omega}_x + (I_z-I_y)\omega_y\omega_z &= \tau_x \\
  I_y\dot{\omega}_y + (I_x-I_z)\omega_z\omega_x &= \tau_y \\
  I_z\dot{\omega}_z + (I_y-I_x)\omega_x\omega_y &= \tau_z
  \end{aligned}
  \]
- 角加速度：\(\dot{\boldsymbol{\omega}} = \mathbf{I}^{-1}(\boldsymbol{\tau} - \boldsymbol{\omega} \times \mathbf{I}\boldsymbol{\omega})\)。

### 2.3 运动学方程（四元数形式）
\[
\dot{q} = \frac{1}{2}\, q \otimes (0, \boldsymbol{\omega})
\]
分量形式（直接用于数值积分）：
\[
\begin{aligned}
\dot{q}_w &= \frac{1}{2}(-q_x\omega_x - q_y\omega_y - q_z\omega_z)\\
\dot{q}_x &= \frac{1}{2}( q_w\omega_x + q_y\omega_z - q_z\omega_y)\\
\dot{q}_y &= \frac{1}{2}( q_w\omega_y - q_x\omega_z + q_z\omega_x)\\
\dot{q}_z &= \frac{1}{2}( q_w\omega_z + q_x\omega_y - q_y\omega_x)
\end{aligned}
\]

### 2.4 数值积分（欧拉法）
- 更新角速度：\( \boldsymbol{\omega}_{k+1} = \boldsymbol{\omega}_k + \dot{\boldsymbol{\omega}}_k \Delta t \)
- 更新四元数：\( q_{k+1} = q_k + \dot{q}_k \Delta t \)
- **关键**：每次更新后归一化四元数 \( q \leftarrow q / \|q\| \)。

## 三、惯量张量（矩阵）的直观理解

- 对角元 \( I_{xx}, I_{yy}, I_{zz} \)：绕 X、Y、Z 轴的转动惯量（单轴熟知的量）。
- 非对角元（惯性积）表示质量分布不对称引起的耦合，若取主轴对齐则全为零。
- 三维动力学中多出的 \( \boldsymbol{\omega}\times(\mathbf{I}\boldsymbol{\omega}) \) 项就是陀螺耦合：即使不受外力矩，若角速度不沿主轴，角速度也会变化。

```python
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
import matplotlib.animation as animation

# ---------- 四元数工具 ----------
def quat_multiply(q1, q2):
    w1,x1,y1,z1 = q1; w2,x2,y2,z2 = q2
    return np.array([
        w1*w2 - x1*x2 - y1*y2 - z1*z2,
        w1*x2 + x1*w2 + y1*z2 - z1*y2,
        w1*y2 - x1*z2 + y1*w2 + z1*x2,
        w1*z2 + x1*y2 - y1*x2 + z1*w2
    ])

def quat_conjugate(q):
    return np.array([q[0], -q[1], -q[2], -q[3]])

def quat_normalize(q):
    return q / np.linalg.norm(q)

def quat_to_euler(q):
    w,x,y,z = q
    roll = np.arctan2(2*(w*x + y*z), 1-2*(x*x + y*y))
    pitch = np.arcsin(2*(w*y - z*x))
    yaw = np.arctan2(2*(w*z + x*y), 1-2*(y*y + z*z))
    return np.array([roll, pitch, yaw])

def axis_angle_to_quaternion(axis, angle):
    half = angle/2.0; s = np.sin(half)
    return np.array([np.cos(half), s*axis[0], s*axis[1], s*axis[2]])

def quaternion_to_rotation_matrix(q):
    w,x,y,z = q
    return np.array([
        [1-2*y*y-2*z*z, 2*x*y-2*z*w, 2*x*z+2*y*w],
        [2*x*y+2*z*w, 1-2*x*x-2*z*z, 2*y*z-2*x*w],
        [2*x*z-2*y*w, 2*y*z+2*x*w, 1-2*x*x-2*y*y]
    ])

# ---------- 卫星动力学类 ----------
class Satellite:
    def __init__(self, I=np.diag([1.2, 1.0, 0.8])):
        self.I = I
        self.I_inv = np.linalg.inv(I)
        self.q = np.array([1.0, 0.0, 0.0, 0.0])   # 初始无旋转
        self.omega = np.zeros(3)

    def set_initial_attitude(self, roll_deg, pitch_deg, yaw_deg):
        q_yaw = axis_angle_to_quaternion([0,0,1], np.radians(yaw_deg))
        q_pitch = axis_angle_to_quaternion([0,1,0], np.radians(pitch_deg))
        q_roll = axis_angle_to_quaternion([1,0,0], np.radians(roll_deg))
        self.q = quat_multiply(q_roll, quat_multiply(q_pitch, q_yaw))

    def set_initial_omega(self, omega_deg_s):
        self.omega = np.radians(omega_deg_s)

    def dynamics(self, torque):
        I_omega = self.I @ self.omega
        omega_dot = self.I_inv @ (torque - np.cross(self.omega, I_omega))
        return omega_dot

    def kinematics(self):
        w,x,y,z = self.q
        wx,wy,wz = self.omega
        return 0.5 * np.array([
            -x*wx - y*wy - z*wz,
             w*wx + y*wz - z*wy,
             w*wy - x*wz + z*wx,
             w*wz + x*wy - y*wx
        ])

    def update(self, torque, dt):
        self.omega += self.dynamics(torque) * dt
        self.q += self.kinematics() * dt
        self.q = quat_normalize(self.q)
        return self.q, self.omega

# ---------- 四元数误差 PID ----------
def quaternion_error(q_target, q_current):
    qc_conj = quat_conjugate(q_current)
    return quat_multiply(q_target, qc_conj)

class QuaternionPID:
    def __init__(self, Kp=8.0, Ki=0.5, Kd=2.0, max_torque=2.0, dt=0.01):
        self.Kp, self.Ki, self.Kd = Kp, Ki, Kd
        self.max_torque = max_torque
        self.dt = dt
        self.integral = np.zeros(3)
        self.prev_error = np.zeros(3)

    def compute(self, q_target, q_current, omega_current):
        q_err = quaternion_error(q_target, q_current)
        error_vec = 2 * q_err[1:4]           # 近似角度误差（小角度）
        self.integral += error_vec * self.dt
        derivative = -omega_current
        torque = (self.Kp * error_vec +
                  self.Ki * self.integral +
                  self.Kd * derivative)
        return np.clip(torque, -self.max_torque, self.max_torque)

# ---------- 仿真主程序 ----------
def run_simulation():
    dt = 0.01; duration = 10.0
    steps = int(duration/dt)
    time = np.linspace(0, duration, steps)

    sat = Satellite()
    sat.set_initial_attitude(30, 20, 10)      # 初始：滚转30°，俯仰20°，偏航10°
    sat.set_initial_omega([0,0,0])
    q_target = np.array([1.0,0,0,0])          # 目标：零姿态

    pid = QuaternionPID(Kp=8.0, Ki=0.5, Kd=2.0, max_torque=2.0, dt=dt)

    euler_deg = np.zeros((steps,3))
    torque_log = np.zeros((steps,3))
    omega_deg = np.zeros((steps,3))
    q_log = np.zeros((steps,4))

    for i in range(steps):
        euler_rad = quat_to_euler(sat.q)
        euler_deg[i] = np.degrees(euler_rad)
        omega_deg[i] = np.degrees(sat.omega)
        q_log[i] = sat.q

        torque = pid.compute(q_target, sat.q, sat.omega)
        torque_log[i] = torque
        sat.update(torque, dt)

    # 绘图（省略详细代码，见原回答）
    # ...

if __name__ == "__main__":
    run_simulation()
```

## 五、今日要点总结

| 概念 | 核心公式/操作 |
|------|----------------|
| 四元数轴角构造 | \( q = (\cos\frac{\alpha}{2},\ \sin\frac{\alpha}{2}\,\mathbf{u}) \) |
| 旋转向量 | \( \mathbf{v}' = q \otimes (0,\mathbf{v}) \otimes q^* \) |
| 姿态更新（左乘） | \( q_{\text{new}} = q_{\text{rot}} \otimes q_{\text{cur}} \) |
| 运动学 | \( \dot{q} = \frac{1}{2} q \otimes (0,\omega) \) |
| 动力学 | \( \dot{\omega} = I^{-1}(\tau - \omega \times I\omega) \) |
| 数值积分 | 欧拉法 + 四元数归一化 |
